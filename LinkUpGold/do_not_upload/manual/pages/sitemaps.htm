Sitemap

This is an optional feature and can be enabled/disabled in Configuration. To enable it, enter a value to the field "Server location of sitemap" in Configuration. This functions creates a page with selected items (categories, links, articles, videos, news), it builds it on each daily rebuild. This page is based on template sitemap.html. 

Sitemap for Google

It creates a XML file that can be read by Google Sitemaps. You can build this page in Administration => Reset/rebuild. Make sure to configure location of this file and also its features - these options are available in Configuration. Each item in this Sitemap is based on the template google_sitemap.txt. 

Sitemap for Yahoo

It creates a TXT file that can be read by Yahoo. You can build this page in Administration => Reset/rebuild. Make sure to configure location of this file and also its features - these options are available in Configuration. Once the it has been created, you can submit it to Yahoo for review.
