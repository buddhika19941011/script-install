List of Main Functions

Links - one category A page which contains advertising links at the top and free links below them, list of next/previous pages, options to sort the selected category by multiple ways, RSS news related to the selected category, subcategories, similar categories, AdLinks etc.

Link details page This page contains all details about the selected link, rating form and list of rates sent in the past, images, user comments, list of other links of the same owner, link owner contact form, list of previous/next links in the same category and in the whole directory.

New link submission Anyone who wants to submit a new link to your directory can do it by this form.

Changes in an existing link Link owners can use this form to update their links.

Articles - one category A page which contains articles of a selected category, list of next/previous pages, options to sort the selected category by multiple ways, RSS news related to the selected category, subcategories, similar categories etc.

Article - details page This page contains all details about the selected article, rating form and list of rates sent in the past, images, user comments, list of other articles of the same owner, article owner contact form, list of previous/next articles in the same category and in the whole directory.

Blogs - one category A page which contains blogs of a selected category, list of next/previous pages, options to sort the selected category by multiple ways, RSS news related to the selected category, subcategories, similar categories etc.

Blogs - details page This page contains all details about the selected blog, rating form and list of rates sent in the past, images, user comments, list of other blogs of the same owner, blogs owner contact form, list of previous/next blogs in the same category and in the whole directory.

Videos - one category A page which contains videos of a selected category, list of next/previous pages, options to sort the selected category by multiple ways, RSS news related to the selected category, subcategories, similar categories etc.

Video - details page This page contains all details about the selected video, rating form and list of rates sent in the past, images, user comments, list of previous/next videos in the same category and in the whole directory.

News - one category A page which contains news of a selected category, list of next/previous pages, options to sort the selected category by multiple ways, RSS news related to the selected category, subcategories, similar categories etc.

News item - details page This page contains all details about the selected news item, rating form and list of rates sent in the past, images, user comments, list of previous/next news in the same category and in the whole directory.

AdLinks Users can order small ads with a few lines of text that are in the right column of categories of links and search results

How to add an article Anyone who wants to submit a new article to your directory can do it by this form..

How to update an existing article Article owners can use this form to update their articles.

Searching in links, articles, blogs, videos and news Simple and advanced search forms are available.

Comments to links, articles, blogs, videos and news Your visitors can add their own comments to each item in your directory. These comments are displayed on the item detail pages.

Advertising links Registered users can add extra features to their links and pay for it. Advertising links are always at the top of categories and search results. Multiple payment options are available.

Message board Simple message board. Your visitors can add their own messages.

Page.php Pages with your own custom content.

Registered users There are many features that are available for registered users only.

Sitemaps Two sitemaps can be created automatically. A public sitemap for your visitors and a sitemap for Google.

RSS output Automatically generated output in RSS format. It can be used in a RSS reader software.
