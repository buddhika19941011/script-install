Link details page - link.php

This script is used to show detailed info about a single link. This script has to be called as link.php?n=LINK_NUMBER where LINK_NUMBER is number of the required link. To show the page is used template link_details.html by default, however you can create any number of your own templates and in the setting of each category set up different templates for different categories. Names of your own templates must look like: link_details*.html where the * will be any English letters, numbers and/or this character: _. The maximum allowed number of characters in names of your own templates is 40, incl. dot and extension. If you will create your own templates, make sure the default template link_details.html always exists, otherwise you get an error.
Sponsored links always use the template link_details_advertising.html.
Used variables:

#%in1%#
#%in2%# 

Content of files _in1.txt .... . This may be used to insert some information to several or all pages. In these files may be also used some variables. More info here.

#%head%#

Content of file _head2.txt. This may be used to insert some information to the begin of several or all pages. More info here.

#%site_url%#

URL of your site

#%n%#

Number of the link.

#%title%#

Title of the link.

#%description%#

Description of the link.

#%thumbnail%#

Thumbnail of the link URL.

#%tags%#

Keywords of the link and category name where is listed this link. Each of these tags contain a link to search results for it.

#%detail%#

Detailed description of the link.

#%user_defined%#

User Defined Items. Each line of this variable is based on template user_item_listing.txt.

#%user_item_28%#

If you, for some reasons, don't want to display all available User Defined Items, use this variable to display each individual item. This is only an example, replace the number 28 by real number of the item you want to display. These numbers can be found in Configuration => User Defined Items.

#%user_item_name_28%#

Use this variable to display name (label) of an individual user defined item.

#%user_item_value_28%#

Use this variable to display value of an individual user defined item.

#%created%#

Date created.

#%updated%#

Date updated.

#%rating%#

Current rating of the link.

#%rateicon%#

Relevant icon of the actual rating. These icons has to be stored in directory 'images' and their names must be '1.gif' for rating 0.5, '2.gif' for rating 1, ...... '10.gif' for rating '5'. If you will use other file formats (jpg or png) for these images, you have to set it in your Configuration (in the Administration).

#%votes%#

Number of votes for the link.

#%hits_m%#

Number of visitors clicked on the link since the last monthly reset.

#%clicks_in_m%#

Number of incoming hits from this link since the last monthly reset. If you want to count these incoming hits, you have to tell the webmasters who submit their links that they have to send hits to click.php?n=LINK_NUMBER where LINK_NUMBER will be actual number of the link.

#%hits%#

Total number of visitors clicked on the link.

#%clicks_in%#

Total number of incoming hits from this link. If you want to count these incoming hits, you have to tell the webmasters who submit their links that they have to send hits to click.php?n=LINK_NUMBER where LINK_NUMBER will be actual number of the link.

#%categories%#

List of categories where is listed selected link.

#%categories_incl%#

List of categories where is listed selected link, this list contains full path of each category incl. all parent categories.

#%pictures_gallery%#

If the link has images, this variable displays previews of all of them.

#%map%#

If the link has an included map, it shows it.

#%rss_content%#

If the link has a RSS URL to show news from the listed site, it shows these news.

#%comments%#

Number of comments received by the link.

#%show_comments%#

Comments for the selected link. Each comment is based on template comment.txt.

#%enter_comment_box%#

A form to enter a comment. It uses the template comment_form.txt.

#%more_items%#

List to other links entered by the same person (email address).

#%next_category%#
#%previous_category%#
#%next%#
#%previous%#

Links to next/previous links and next/previous links in the same category.

#%icons%#

If the link is New, Popular, Updated or Editor Pick it shows the relevant icon(s).

#%tell_friend_box%#

A form to email a friend about this link. It uses the template tell_friend.txt.

#%report_box%#

A form to inform admin about an error. It uses the template report.txt.

#%notes_edit_box%#

A form to enter private notes (available for registered users only). It uses the template notes_form.txt.

#%notes%#

Entered private notes (available for registered users only).

#%contact_box%#

A form to contact the link owner. It uses the template contact_form.txt.

#%currency%# #%price%#

Currency and price for each click. This is applicable for advertising links with dynamic prices.

#%add_delete_favorites%#

The option to add or remove the link from user's favorites (available for registered users only).

#%edit_link%#

A link to a link edit page. This is available for admins only.

#%link%#

 If the link has a URL, it shows its title linked to this URL. If it doesn't have a URL, it shows link title only.

#%hide_something_begin%#
#%hide_something_end%# 

There are multiple variables which look like this example. These variables are used to hide content which is not applicable for the current category or state. You should not edit these variables.
