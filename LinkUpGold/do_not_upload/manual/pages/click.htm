Incoming clicks - click.php

This script counts incoming clicks for each link and then sends the visitor to your home page or to the category where the link comes from - both these options are in Configuration. If you want to count incoming clicks you have to ask link owners to send clicks to click.php?n=LINK_NUMBER where LINK_NUMBER is an actual number of a link.
Then you can set that links should be sorted by incoming clicks so links sending more clicks will be higher up on your pages.
Note: Link owners also can use your search box and each searching they send will be counted as one hit sent to you. More info here.
