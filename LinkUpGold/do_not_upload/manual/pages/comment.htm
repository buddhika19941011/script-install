User comments - comment.php

This script is used to add a comment to a link or article or blog or video or news item. The form is available in categories in the listing and also on detail pages of each item. It uses the template comment_form.txt. The list of comments for each item is available on the same places. There is used the template comments.txt for the list of comments and comment.txt for each of them.

To inform admin that a comment has been added is used email template comment_added.txt.

Once a comment has been submitted, it can be in the queue so admin has to review it or it can be added directly to the online database. These options are in Administration => Public forms.
