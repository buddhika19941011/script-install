Paypal.com

The system supports automatic payments by Paypal. Your users pay and the system can detect that a payment has been made. To get it to work you need to allow IPN in Paypal users area. You also need to have a business account because this feature is not available for personal accounts. Also enter your primary Paypal email address in Link Up Gold Configuration and enter values to the other fields that are related to Paypal. No further configuration is needed. You can change the image used in Paypal button by editing template order_form_paypal.txt.

2CheckOut.com

The system supports automatic payments by 2Checkout. Your users pay and the system can detect that a payment has been made. To get it to work enter values to the fields in Configuration that are related to 2CheckOut. You also need to go 2CheckOut users area and click to the link "Look & Feel Settings" link. Find these fields on the page you receive:

  • Approved URL and Pending URL. Enter http://www.yourdomain.com/payment_2checkout.php to both these fields (replace http://www.yourdomain.com by your real URL).
  • Direct Return select box. Select "No".
  • Secret Word - enter there anything you want, you should use a strong word so nobody can find it out. Then copy this word to Link Up Gold Configuration - there is the field "Secret word" in the section "2CheckOut data".
