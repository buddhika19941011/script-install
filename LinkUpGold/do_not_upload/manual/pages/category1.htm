Category of links - links.php

This script is used to display pages of all categories of links.

To show a single category you will need to call the script as links.php?n=CATEGORY_NUMBER. It uses template category.html by default, however you can create any number of your own templates and in the setting of each category set up different templates for different categories. Names of your own templates must look like: category*.html where the * are any English letters, numbers and/or this character: _. The maximum allowed number of characters in names of your own templates is 40, incl. dot and extension. If you will create your own templates, make sure the default template category.html still exists, otherwise you receive an error. Used variables:

#%in1%#
#%in2%# 

Content of files _in1.txt .... . This may be used to insert some information to several or all pages. In these files may be also used some variables. More info here.

#%head%#

Content of file _head2.txt. This may be used to insert some information to the begin of several or all pages. More info here.

#%site_url%#

URL of your site

#%name%#

Name of the current category.

#%description%#

Description of the current category.

#%n%#

Number of the current category.

#%arrow%#

It creates an "arrow" so visitors can see where they are. The arrow looks like: Home > First Level Category > Parent Category >

#%ad1%#
#%ad2%#
#%ad3%# 

Ads defined for the selected category.

#%image%#

Image defined for the category. You can set different image for each category in your Administration.

#%item_submit_url%#
#%item_submit_text%#

URL and text of a link to the item submit page. It's displayed only if it's allowed to submit items to the selected category.

#%pages%#

Form or links which let your users skip to next page(s) of the same category or sort links by another way. This form is based on template pages_form.txt or pages_list.txt, based on your configuration.

#%user_defined%#

User Defined Items. Each line of this variable is based on template user_item_listing.txt.

#%user_item_28%#

If you, for some reasons, don't want to display all available User Defined Items, use this variable to display each individual item. Content of this variable is based on template user_item_listing.txt.
This is only an example, replace the number 28 by real number of the item you want to display. These numbers can be found in Configuration => User Defined Items.

#%user_item_name_28%#

Use this variable to display name (label) of an individual user defined item.

#%user_item_value_28%#

Use this variable to display value of an individual user defined item.

#%subcategories%#

If the current category has subcategories, it makes a table with links to these subcategories and numbers of links in each subcategory. The table is based on template categories.txt, each subcategory in this table is based on template categories_category.txt.

#%items%#

List of links in the current category. Each link is based on template link_a.txt - more info below.

#%similar%#

If the current category has defined some similar categories, it shows a list of these similar categories.

#%rss_content%#

RSS content defined for the current category. You can enter a RSS URL for each category in the category edit form in Administration.

#%add_delete_favorites%#

The option to add or remove the category to/from user's favorites (available for registered users only).

#%notes%#

It shows private notes for the selected category (available for registered users only).

#%notes_edit_box%#

A form to edit private notes for the selected category (available for registered users only). This form is based on template notes_form.txt.

#%edit_link%#

A link to a category edit page. This is available for admins only.

#%adlinks%#

It shows AdLinks defined for the selected category. Registered users can order AdLinks in users area. You can define the prices and look and feel of AdLinks in Administration.

#%hide_something_begin%#
#%hide_something_end%# 

There are multiple variables which look like this example. These variables are used to hide content which is not applicable for the current category or state. You should not edit these variables.


Each link is based on template link_a.txt by default, however you can create any number of your own templates and in the category edit form set up different templates for different categories. Names of your own templates must look like: link_a*.txt where the * will be any English letters, numbers and/or this character: _. The maximum allowed number of characters in names of your own templates is 40, incl. dot and extension. If you will create your own templates, make sure the default template link_a.txt always exists, otherwise you get an error. Sponsored links are not based on the template link_a.txt but on template link_advertising.txt. These templates are also used to show links in search results and in special categories (new links etc.).

These variables may be used in both these templates:

#%n%#

Number of the link.

#%url%#

Link URL.

#%title%#

Title of the link.

#%description%#

Description of the link.

#%user_defined%#

User Defined Items. Each line of this variable is based on template user_item_listing.txt.

#%user_item_28%#

If you, for some reasons, don't want to display all available User Defined Items, use this variable to display each individual item. Content of this variable is based on template user_item_listing.txt.
This is only an example, replace the number 28 by real number of the item you want to display. These numbers can be found in Configuration => User Defined Items.

#%user_item_name_28%#

Use this variable to display name (label) of an individual user defined item.

#%user_item_value_28%#

Use this variable to display value of an individual user defined item.

#%notes%#

Private notes related to this link. This is available for registered users only.

#%icons%#

If the link is New, Popular, Updated or Editor Pick it shows the relevant icon(s).

#%tags%#

Keywords of the link and category name where is listed this link. Each of these tags contain a link to search results for it.

#%item_details_url%#

URL of the page with link details.

#%comments%#

Number of comments received by the link.

#%add_delete_favorites%#

The option to add or remove the link from user's favorites (available for registered users only).

#%image_1%#

If the link has images, this variable shows the first of them.

#%currency%# #%price%#

Currency and price for each click. This is applicable for advertising links with dynamic prices.

#%edit_link%#

A link to a link edit page. This is available for admins only.

#%link%#

 If the link has a URL, it shows its title linked to this URL. If it doesn't have a URL, it shows link title only.

#%created%#

Date created.

#%updated%#

Date updated.

#%rating%#

Current rating of the link.

#%rateicon%#

Relevant icon of the actual rating. These icons has to be stored in directory 'images' and their names must be '1.gif' for rating 0.5, '2.gif' for rating 1, ...... '10.gif' for rating '5'. If you will use other file formats (jpg or png) for these images, you have to set it in your Configuration (in the Administration).

#%votes%#

Number of votes for the link.

#%hits_m%#

Number of visitors clicked on the link since the last monthly reset.

#%clicks_in_m%#

Number of incoming hits from this link since the last monthly reset. If you want to count these incoming hits, you have to tell the webmasters who submit their links that they have to send hits to http://www.yourdomain.com/click.php?n=LINK_NUMBER where LINK_NUMBER will be actual number of the link.

#%hits%#

Total number of visitors clicked on the link.

#%clicks_in%#

Total number of incoming hits from this link. If you want to count these incoming hits, you have to tell the webmasters who submit their links that they have to send hits to http://www.yourdomain.com/click.php?n=LINK_NUMBER where LINK_NUMBER will be number of the link.

#%hide_something_begin%#
#%hide_something_end%# 

There are multiple variables which look like this example. These variables are used to hide content which is not applicable for the current category or state. You should not edit these variables.

#%enter_comment_box%#

A form to enter a comment. It uses the template comment_form.txt.

#%tell_friend_box%#

A form to email a friend about this link. It uses the template tell_friend.txt.

#%report_box%#

A form to inform admin about an error. It uses the template report.txt.

#%notes_edit_box%#

A form to enter private notes (available for registered users only). It uses the template notes_form.txt.

#%notes%#

Entered private notes (available for registered users only).
