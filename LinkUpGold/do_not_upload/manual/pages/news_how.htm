News

News can be imported from rss sources. Your directory can receive a new content many times a day without any effort. In the category edit forms you can set how often it should import news to each of your categories. It's also possible to use a crontab command to run these imports. The command should look exactly like the command which runs daily job, only add this to the end of the URL in the command:

&action=import_rss&c=CATEGORY_NUMBER

Instead of CATEGORY_NUMBER use real number of the category where it should import news. Each category should have its own crontab command.
