HTML Plugin

HTML Plugin is an optional addon for Link Up Gold. It can be ordered in our users area. This plugin has two main options. You can switch between them in Configuration - at the end of the form there is HTML Plugin section. These options are:

  • Static HTML Pages If this option is enabled, your home page, all categories and optionally link and article detail pages are static. The URL's are search engines friendly and it also saves server resources because it doesn't have to read the database each time when someone browses your site.
