Video details page - video.php

This script is used to show detailed info about a single video. This script has to be called as video.php?n=VIDEO_NUMBER where VIDEO_NUMBER is number of the selected video. To show this page is used template video_details.html by default, however you can create any number of your own templates and in setting of each category set up different templates for different categories. Names of your own templates must look like: 'video_details*.html' where the * will be any English letters, numbers and/or this character: _. The maximum allowed number of characters in names of your own templates is 40, incl. dot and extension. If you will create your own templates, make sure the default template video_details.html always exists, or you receive an error. Available variables:

#%in1%#
#%in2%# 

Content of files _in1.txt .... . This may be used to insert some information to several or all pages. In these files may be also used some variables. More info here.

#%head%#

Content of file _head2.txt. This may be used to insert some information to the begin of several or all pages. More info here.

#%site_url%#

URL of your site

#%n%#

Number of the video.

#%title%#

Title of the video.

#%description%#

Subtitle of the video.

#%text%#

Text of the video.

#%tags%#

Keywords of the video and category name where is listed this video. Each of these tags contain a link to search results for it.

#%user_defined%#

User Defined Items. Each line of this variable is based on template user_item_listing.txt.

#%user_item_28%#

If you, for some reasons, don't want to display all available User Defined Items, use this variable to display each individual item. This is only an example, replace the number 28 by real number of the item you want to display. These numbers can be found in Configuration => User Defined Items.

#%user_item_name_28%#

Use this variable to display name (label) of an individual user defined item.

#%user_item_value_28%#

Use this variable to display value of an individual user defined item.

#%created%#

Date created.

#%updated%#

Date updated.

#%rating%#

Current rating of the video.

#%rateicon%#

Relevant icon of the actual rating. These icons has to be stored in directory 'images' and their names must be '1.gif' for rating 0.5, '2.gif' for rating 1, ...... '10.gif' for rating '5'. If you will use other file formats (jpg or png) for these images, you have to set it in your Configuration (in the Administration).

#%votes%#

Number of votes for the video.

#%hits%#

Number of readers if the video.

#%hits_m%#

Number of readers if the video the current month.

#%pictures_gallery%#

If the video has images, this variable displays previews of all of them.

#%map%#

If the video has an included map, it shows it.

#%comments%#

Number of comments received by the video.

#%show_comments%#

Comments for the selected video. Each comment is based on template comment.txt.

#%more_items%#

List to other videos entered by the same person (email address).

#%next_category%#
#%previous_category%#
#%next%#
#%previous%#

Links to next/previous videos and next/previous videos in the same category.

#%icons%#

If the video is New, Popular, Updated or Editor Pick it shows the relevant icon(s).

#%tell_friend_box%#

A form to email a friend about this video. It uses the template tell_friend.txt.

#%notes_edit_box%#

A form to enter private notes (available for registered users only). It uses the template notes_form.txt.

#%notes%#

Entered private notes (available for registered users only).

#%add_delete_favorites%#

The option to add or remove the video from user's favorites (available for registered users only).

#%edit_link%#

A link to an video edit page. This is available for admins only.

#%hide_something_begin%#
#%hide_something_end%# 

There are multiple variables which look like this example. These variables are used to hide content which is not applicable for the current category or state. You should not edit these variables.
