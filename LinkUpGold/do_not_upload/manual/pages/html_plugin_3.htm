HTML Plugin - Apache Rewrite Module

All the pages are dynamic, all the common features can be used as before. This option creates nice / search engine friendly URL's. Example:

  • URL of your home page is http://www.yourdomain.com/index.html instead of http://www.yourdomain.com/index.php.
  • URL's of categories look like http://www.yourdomain.com/directory-articles/category-name/
  • URL's for individual links/articles/news/videos look like http://www.yourdomain.com/directory-link-ITEM_NUMBER/link-title.html

To enabled this option check the field "Use Apache Rewrite module" in Configuration. Also enter values to these fields (each of them should have an unique value):

  • Virtual folders to store pages
  • Extension of links/articles/videos/news detail pages

This option needs to have Rewrite Module enabled/allowed in Apache configuration. It also needs you to create a text file in Link Up Gold main directory. Name of this file will be ".htaccess". Note that name of this file has a dot at the begin. The commands for this line will be generated automatically in Configuration, at the bottom of the form. Copy these commands to your .htaccess file. On the end of this file must be one blank line.

When you use this option, use only classic links to link to previous/next pages. Do not use the drop-down menu option.
