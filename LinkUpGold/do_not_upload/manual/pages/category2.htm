Extra categories of links - links.php

Extra categories are New Links, Popular Links, Editor Picks and Top Rated Links. Default URL's of these categories are as follows:
New Links links.php?action=new
Editor Picks links.php?action=pick
Top Rated Links links.php?action=top_rated
Popular Links links.php?action=popular

The template used for these pages is category_special.html. There are available the following variables:

#%in1%#
#%in2%# 

Content of files _in1.txt .... . This may be used to insert some information to several or all pages. In these files may be also used some variables. More info here.

#%head%#

Content of file _head2.txt. This may be used to insert some information to the begin of several or all pages. More info here.

#%site_url%#

URL of your site

#%title%#

Page title. You can define these titles in Administration => Messages.

#%items%#

Links in the current category. Each link is based on template link_a.txt (free links) or link_advertising.txt (advertising links).
