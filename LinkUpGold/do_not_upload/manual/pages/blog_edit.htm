Editing an existing blog - blog_edit.php

This script is used to edit an existing blog.

To show the form where a blog owner can login to edit a blog should be the script called as blog_edit.php. This login form is applicable only for blog owners who are not registered users. Registered users can see a list of their blogs inside users area, they can edit each of these blogs without the request to enter an additional password.

The login form is based on template blog_edit_login.html. Used variables:

#%in2%#

#%head%#

Content of file _head2.txt. This may be used to insert some information to the begin of several or all pages. More info here.

#%site_url%#

URL of your site.

#%info%#

If the blog owner just asked to send a blog password to his/her email, it shows this info.

#%n%#

If the blog owner just asked to send a blog password to his/her email, it shows number of the selected blog.

If the blog owner doesn't know info about the blog which he/she wants to edit, the script can show a list of all blogs associated to an entered email address. There is a field to enter email in the login form. The list of blogs that are associated to an email is based on template blog_edit_list.html - the list of blogs associated with the entered email. This list contains links to blog details page, link to a login form to edit it and a blog to password reminder. Each of the lines is based on template blog_edit_list.txt.

Once a blog has been edited, multiple states can occur, depending on your configuration. These are:

  • You selected that it has to inform admin when a link has been edited. In this case you receive an email by using template blog_admin.txt.
  • You selected that updated blogs must be reviewed by an admin. In this case the blog is waiting in Administration => Queue. Once you approved it, the blog owner receives an email based on template blog_approved.txt.
  • You selected that edited blogs are immediately online. In this case the old data has been deleted and the edited blog is instantly visible on public pages.
