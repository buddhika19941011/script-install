General page with your own content - page.php

This script can be useful to create pages which look like all the other pages in your directory, these pages can have your own content. You have to call this script as page.php?page=TEMPLATE_NAME where TEMPLATE_NAME is name of a template which should be used, without extension. The extension of the template must be .html. For example if name of the template is my_page.html, you will call this script as page.php?page=my_page. You can create any number of different custom pages. There is template page.html available in templates directory, you can use it as an example of these custom pages.
