Installation

1) Extract the zip which contains Link Up Gold. It creates a folder, feel free to rename it.

2) Upload all files on your server into the correct directories. Upload all the files but images in ASCII mode. Do not upload folder "do_not_upload".

3) Set permissions for directories and files.
This step is needed on some servers only. You can skip it and follow with step #4. If you will receive a permission error, follow also this step #3.

  • These files and directories need permission 777:
    - data directory
    - messages directory (it's inside "styles/_common" directory)
    - email_templates directory (it's inside "styles/_common" directory)
    - templates directory (it's inside "styles/_common" directory and also inside some other subdirectories of the styles directory)
    - images/articles directory
    - images/categories directory
    - images/links directory
    - images/news directory
    - images/users directory
    - images/videos directory
    - images/blogs directory
    - uploads/qrcodes folder
    - files sitemap.txt, sitemap.html and sitemap.xml in main directory
  • These files need permission 666:
    - file "data_forms.php" in data directory
    - all files in the "messages" directory (it's inside "styles/_common" directory)
    - all files in "email_templates" directory (it's inside "styles/_common" directory)
    - all files in "templates" directories (it's inside "styles/_common" directory and also inside some other subdirectories of the styles directory)
  • These directories need permission 755 (it usually doesn't need any change, servers usually set it automatically):
    - directory where your scripts and other directories reside
    - administration directory
    - images directory

4) Point your browser to your domain to file setup.php. Now fill in all required fields and click to the Install button. You should see that the script creates tables and files for Link Up Gold.

5) Once the script has been successfully installed, go to administration and follow these steps:

  • Remove setup.php from your server.
  • Main configuration - enter values to all fields. Save the form.
  • Reset/rebuild - run Daily Job. It builds all pages. If you receive an error during this step, there is some missing value in Configuration.
  • Add a few categories to test if all works well. To make these items visible on the home page and in the left and right columns of all pages, run the Daily Job again.
  • Set the script up to run Daily Job. Read more about this issue here.
  • You also can customize templates. More info here here.
