Advertising/sponsored links

Registered users can order extra features for their links and pay for it. We call such links as Advertising or Sponsored links. Free links use templates link_a.txt and link_details.html, while advertising links use templates link_advertising.txt and link_details_advertising.html. It means that you can mark advertising links by some way (bigger font, another color etc.). Styles used in these paid links are highlighted by default.

Advertising links are always at the top of categories and search results. You also can configure the system so advertising links can have advantages like more pictures, the option to submit them to more categories, etc.

There is not an extra form to create an advertising link. Users can create normal free links and then upgrade them to advertising links. The process of free link submission is described here.

Users who want to see their links can use the link "Your links & invoices" in the menu in users area. URL of the page is link_edit.php?action=links_list. This page uses template links_for_owner_list.html. The first table on this page shows all existing links of the user. Existing links are displayed by the variable #%links%#, each line in this list is based on template links_for_owner_list_link.txt. This template contains the following variables:

#%site_url%#

URL of your site.

#%n%#

Number of an link.

#%created%#

Date when the link has been created.

#%item_details_url%#

URL of public link details page.

There is the link "Extra options" available for each link. The URL of the page is link_extra_features.php?action=link_adv_home&n=LINK_NUMBER. This page offers all the options that make the link sponsored - there are options to add advertising days, clicks and impressions to it. This page uses template link_extra_features.html. This page offers two main options:

  • Simple option - User orders a limited number of days, pays by a credit card and the link becomes immediately sponsored or it adds new days to it if the link was sponsored before. The order process is described here.
  • Flexible options - These options use funds (money) prepaid before. These funds are on users account and can be used for advertising links and AdLinks. Your users can use packages you created in Administration => Packages to add new funds to their accounts. The order process is described here. These flexible options allow users to order clicks, impressions or days, the prices for all these are editable in Configuration. If they want to order clicks, they can choose between fixed prices (in this case prices for all clicks are the same) or dynamic prices (in this case you set a range of prices and users can choose a price for each click inside this range, more expensive links are higher).
    These flexible options also contain the option to cancel unused clicks, impressions and days. If this is used, the funds are refunded to users main account and can be used for another options or for another link.
