Home page - index.php

This page is based on template index.html. There are available the following variables:

#%in1%#
#%in2%# 

Content of files _in1.txt .... . This may be used to insert some information to several or all pages. In these files may be also used some variables. More info here.

#%head%#

Content of file _head2.txt. This may be used to insert some information to the begin of several or all pages. More info here.

#%site_url%#

URL of your site

#%site_name%#

Name of your site

#%l_categories%#

Table with links to all first level categories with numbers of links in each category. Each field of the table is based on template index_category.txt - more info below. In your configuration you can set how many columns should have this table.

#%a_categories%#

Table with links to all categories of articles with numbers of articles in each category. Each field of the table is based on template index_category.txt - more info below. In your configuration you can set how many columns should have this table.

#%v_categories%#

Table with links to all categories of videos with numbers of videos in each category. Each field of the table is based on template index_category.txt - more info below. In your configuration you can set how many columns should have this table.

#%n_categories%#

Table with links to all categories of news with numbers of news in each category. Each field of the table is based on template index_category.txt - more info below. In your configuration you can set how many columns should have this table.

#%b_categories%#

Table with links to all categories of blogs with numbers of items in each category. Each field of the table is based on template index_category.txt - more info below. In your configuration you can set how many columns should have this table.

#%categories_colspan%#

Number of columns of the table with groups of categories. You have to set this number in Configuration. More info about groups for categories below.

#%time_0%#
...
#%time_23%# 

Messages for individual hours of day. You can edit them in Administration => Configuration => Messages.

#%x_categories_group_1%#
#%x_categories_group_2%#
...
#%x_categories_group_10%# 

Groups of categories. You can assign each category to a group. Then, on the home page you can display these groups. It lets you display categories exactly as you want them, additionally this option let's you display categories of links and categories of articles in the same table/group.

#%l_new_items%#
#%a_new_items%#
#%v_new_items%#
#%n_new_items%#
#%b_new_items%#

Newest links, articles, videos, blogs and news. Each item in these tables is based on template index_new_item.txt.

#%hide_site_news_begin%#
#%hide_site_news_end%# 

HTML tags which hide news. It's used if there are no news available.

#%site_news%#

News. You can enter and edit them in Administration => News.


To show each field of the table which contains list of all first level categories is used the template index_category.txt. These variables can be used in this template:

#%width%#

Cell width

#%folder_icon%#

URL of the category folder. This icon depends on the last time when there has been added the last link or article. You can set these icons in configuration.
Each category also can have its own unique icon which is editable in the category edit form.

#%site_url%#

URL of your site

#%catlink%#

URL of the category

#%name%#

Name of the category

#%items%#

Number of links or articles in this category

#%description%# 

Description of the category

#%subcats%#

List of subcategories of the category
