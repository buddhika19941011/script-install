Error/abuse report - report.php

Your visitors can use this script to inform you about a dead link or a link that abuses your directory by some way. A link that leads to this script is in templates link_a.txt, link_details.html, link_advertising.txt, link_details_advertising.html. It opens a small window with the report form in it.

These error reports are stored in database and you can see them in Administration => Error reports. The system also can send you an email when a new error report has been submitted. This option is in Administration => Main configuration. There is the field "Inform admin when an error report has been submitted".

To show the form to send this report is used template link_report.html. All fields in this form are optional. Used variables:

#%site_url%#

URL of your site

#%url%#

URL of the link.

#%title%#

Title of the link.

#%n%#

Number of the link.

#%info%#

If the visitor tried to send a report, but for some reason (not existing link, wrong email) it has been rejected, it shows the reason.

#%comment%#

If the visitor tried to send a report, but for some reason (not existing link, wrong email) it has been rejected, it shows entered text of the message.

#%name%#

If the visitor tried to send a report, but for some reason (not existing link, wrong email) it has been rejected, it shows entered name.

#%email%#

If the visitor tried to send a report, but for some reason (not existing link, wrong email) it has been rejected, it shows entered email.

#%field_captcha_test%#

It shows CAPTCHA image to test if there is a real live person (optional, can be disabled by admin).


To show a "Thank You" page is used template link_reported.html. Used variables:

#%site_url%#

URL of your site

#%url%#

URL of the link.

#%title%#

Title of the link.

#%comment%#

Text of the report.

#%name%#

Name of the owner.

#%email%#

Email of the owner.


To send an email to admin is used template report.txt. All email templates may be found in "styles/_common/email_templates" directory. In the first line of the template must be word 'Subject:' and then some subject for the email. Then has to be one blank line and then text of the email. Used variables:

#%site_url%#

URL of your site

#%n%#

Number of the link.

#%comment%#

Text of the report.

#%name%#

Name.

#%email%#

Email.

#%url%#

URL of the link.

#%title%#

Title of the link.

#%c_number%#

Comment number
