Searching on public pages - search.php

This script is used to search for links, articles, blogs, videos and news. There are two forms to start searching:

  • Simple search form - it's used to search for links, articles, blogs, news and videos. This form is in template _head2.txt (it's at the top of all pages) and index.html (it's on the front page)
  • Advanced search form (see below).

The simple and advanced search forms searches in Link/Article/Video/News item Title, Description/Subtitle, Detailed Description/Text, Link URL, Keywords. The advanced search form allows you to create a field to search also in a particular user field and even in a range of numeric values. The code for each of your user fields is available in Administration => User defined items. Click on the name of the field you want to include to the advanced search form. There is the edit form and below it is a table with available codes that can be used to search in the selected user field, these codes can be used in the advanced search form in template search.html.

Search Results

To show search results is when searching in the advanced search form or when searching for only links or only articles or only videos or only news in the simple search form is used template search_result.html. These variables are used:

#%in2%# 

Content of files _in1.txt .... . This may be used to insert some information to several or all pages. In these files may be also used some variables. More info here.

#%head%#

Content of file _head2.txt. This may be used to insert some information to the begin of several or all pages. More info here.

#%site_url%#

URL of your site

#%searched%#

Searched phrase(s).

#%result_categories%#

It displays categories whose titles, descriptions or user defined items contain the searched phrases.

#%items%#

List of the links, articles, news or videos that meet the search criteria.

#%adlinks%#

List of the AdLinks that meet the search criteria.

#%pages%#

This variable is based on templates pages_form.txt or pages_list.txt, depending on your configuration. It shows the total number of links or articles in search results, list of pages with the option to skip to other pages and also options to sort the results by another criteria.

RSS Output
The system offers also RSS output for searching. The URL's to call these RSS feeds look as follows:
http://www.yourdomain.com/search.php?rss=PHRASE_TO_SEARCH_FOR
If you use the HTML Plugin with the Rewrite option active, you also can use this URL:
http://www.yourdomain.com/search_rss/test/
