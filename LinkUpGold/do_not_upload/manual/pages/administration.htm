System of Administrators

You can have an unlimited number of administrators. Each administrator can have different permissions. There is even the option to have administrators who can edit links, articles, blogs, videos and news in some categories only. To create such admin check the following field(s) in the Administrators edit form:
- Add/view/edit/delete links
- Add/view/edit/delete articles
- Add/view/edit/delete videos
- Add/view/edit/delete news
- Add/view/edit/delete blogs
Uncheck the following field(s):
- Add/view/edit/delete links of all categories, can import links
- Add/view/edit/delete articles of all categories 
- Add/view/edit/delete videos of all categories 
- Add/view/edit/delete news of all categories 
- Add/view/edit/delete blogs of all categories 
Then you receive another table with a list of categories. There you can choose the categories where the administrator can edit links/articles/blogs/videos/news.

The login system for administrators uses sessions and cookies. It means each administrators must have enabled cookies to log in.

Be very careful if you want to give someone the permission to edit administrators. This person will be able to edit also own account and grab as many permissions as he/she wants.
