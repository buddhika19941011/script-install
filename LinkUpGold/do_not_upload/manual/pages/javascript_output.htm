Javascript output

This feature offers the option to give owners of other sites your code which then can show your content on their sites. All the following tags use the templates javascript_export.txt and javascript_export_item.txt.

The following code can show newly added links:
<script language="javascript" src="http://www.yourdomain.com/js_export.php?what=l"></script>

The following code can show newly added articles:
<script language="javascript" src="http://www.yourdomain.com/js_export.php?what=a"></script>

The following code can show newly added blogs:
<script language="javascript" src="http://www.yourdomain.com/js_export.php?what=b"></script>

The following code can show newly added videos:
<script language="javascript" src="http://www.yourdomain.com/js_export.php?what=v"></script>

The following code can show newly added news:
<script language="javascript" src="http://www.yourdomain.com/js_export.php?what=n"></script>
