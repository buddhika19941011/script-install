RSS output - rss.php

This script can create RSS output. Your visitors can subscribe to your RSS channel and automatically receive new content from your directory to their readers. It may bring you many returning visitors.

How to call this script (which URL should use the visitors who want to join some of your RSS channels):

  • rss.php?action=links_new (new links)
  • rss.php?action=links_pick (links - editors pick)
  • rss.php?action=links_popular (popular links)
  • rss.php?action=l&c=123 (links from category #123, the same is available for all categories)
  • rss.php?action=articles_new (new articles)
  • rss.php?action=articles_pick (articles - editor picks)
  • rss.php?action=articles_popular (popular articles)
  • rss.php?action=a&c=123 (articles from category #123, the same is valid for all categories).
  • rss.php?action=videos_new (new videos)
  • rss.php?action=videos_pick (videos - editor picks)
  • rss.php?action=videos_popular (popular videos)
  • rss.php?action=v&c=123 (videos from category #123, the same is valid for all categories).
  • rss.php?action=news_new (newest news)
  • rss.php?action=news_pick (news - editor picks)
  • rss.php?action=news_popular (popular news)
  • rss.php?action=n&c=123 (news from category #123, the same is valid for all categories).

These links are in template _in1.txt (left column of all pages).

You should create your own image 60x60 pixel and place it to images directory (replace original image rss_logo.gif). Some RSS readers display this image next to your RSS content.

These templates are used to show the RSS content:

  • rss_output.html (the complete page)
  • rss_output.txt (each individual link, article, blog, video, news item).
