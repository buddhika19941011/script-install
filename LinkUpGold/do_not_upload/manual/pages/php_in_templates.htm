Your own PHP code

You can enter your own PHP commands in html templates. If you want to use this feature, allow it in Configuration. Use the tag <? to open php code and ?> to close php code. If you want the php code to print something on pages, don't use echo() command but put your output to variable $line. For example if you want to print the text "This is my text", use this command:

$line .= 'This is my text';

Not all php code will work by this way because there may be a conflict between your code and the code in our scripts, for example there can be the same names of functions or so.
