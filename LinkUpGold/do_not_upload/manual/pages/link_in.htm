Links in frame - link_in_frame.php

This script is used to open a selected link in a frame. Your visitors can see the a link or browse between all links in your directory without leaving your site. You can use it instead of the link URL in the templates which are used to show a link (link_a.txt, link_advertising.txt, link_details.html) by replacing #%url%# with "link_in_frame.php?link=#%n%#". You also can use that as an alternative for your visitors (this way used in the original templates).
The URL to call this script is link_in_frame.php?link=LINK_NUMBER where LINK_NUMBER is number of the required link.

To show the main frame is used template link_in_frame.html, these variables are used:

#%site_url%#

URL of your site.

#%n%#

Number of the link.

#%title%#

Title of the link.

#%c%#

Category number.

To show the top frame is used template link_in_frame_top.html. Used variables:

#%site_url%#

URL of your site

#%n%#

Number of the link.

#%title%#

Title of the link.

#%url%#

URL of the link.

#%item_details_url%#

URL of the link details page.

#%category_url%#

URL of the category where the link comes from.

#%category_name%#

Name of the category where the link comes from.

#%c%#

Number of the category where the link comes from.

#%previous%#
#%next%#
#%previous_category%#
#%next_category%# 

Links to previous/next links in your directory and to previous/next links in the same category.

#%hide_something_begin%#
#%hide_something_end%# 

There are multiple variables which look like this example. These variables are used to hide content which is not applicable for the current link. You should not edit these variables.
