How to insert content of a file to a script (SSI like)

You can insert content of any text-based file into your HTML templates. Place the following code to the place where you want to display your content:
For html or text files: 

<FILE>/full/path/to/a/file.html</FILE>

For scripts: 

<FILE>http://www.domain.com/somescript.php</FILE>

Notes:
All from <FILE> to </FILE> must be on the same line and it must be the only item on this line.
The word FILE is case sensitive, <file> will not work.
