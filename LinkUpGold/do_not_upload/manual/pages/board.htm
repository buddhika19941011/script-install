Message board - board.php

This script creates a simple message board. Your visitors can add their own messages which will be available on your pages.

To show the Board is used template board.html. These variables may be used here:

#%in1%#
#%in2%# 

Content of files _in1.txt .... . This may be used to insert some information to several or all pages. In these files may be also used some variables. More info here.

#%head%#

Content of file _head2.txt. This may be used to insert some information to the begin of several or all pages. More info here.

#%site_url%#

URL of your site

#%info%#

If the visitor just added a message to the Board, this shows an info that the massage has been added.

#%messages%#

All the messages in the Board

To show each message is used template board_message.txt. Here you can use these variables:

#%text%#

Message text

#%link%#

It's a link to send an email to the owner if it was added by an unregistered user. If the message was added by a registered user, it's a link to page with info regarding this user.

#%name%#

Name of owner of the message

#%date%#

Date when the message has been added

Below the message board is available a link which opens a form to add a message. This form uses the template board_add_message.txt.
