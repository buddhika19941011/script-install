Registered users - Introduction

There are many advantages for registered users:

  • Each user has his/her own page inside your site. This page contains details about the selected user, an article entered by the user and also uploaded images. It also contains a wall when each user can post as many messages as needed. If the user owns links, articles or blogs, this page contains all these.
  • Users can ask other users for friendship, these friends are displayed on the user pages.
  • They can choose their favorite default style of your pages.
  • They can join your newsletters.
  • They can submit links, blogs and articles without the need to enter their name, email, password and CAPTCHA image test.
  • They can edit their links, blogs and articles without the need to login again.
  • They can order Advertising links and AdLinks.
  • Only registered users can see detailed statistic of their links.
  • You can choose that only registered users can see categories and/or links/blogs/articles/videos/news detail pages.
  • You can choose that only registered users can submit links and/or blogs and/or articles.
  • You can choose that only registered users can add comments to links/articles/blogs/videos/news.
  • You can choose that only registered users can add messages to the Board.
  • You can choose that only registered users can rate for links, blogs, articles, videos and news.
  • Users can bookmark their favorite links, blogs, articles, videos, news and categories.
