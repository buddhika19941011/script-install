AdLinks

AdLinks are small ads with a few lines of text that are in the right column of categories of links and search results. There can be 1-10 text lines, these are automatically formatted by using template adlink_template.txt (your users enter plain text only) or adlink_template_html.txt (if you allow users to enter plain HTML AdLinks). All these AdLinks are placed to template adlinks.txt.

Users don't pay for impressions but only for clicks. They can choose the price they want to pay for each click, more expensive links are higher. You need to set the minimum and maximum allowed price - there are fields for it in Configuration.

The option to create a new AdLink is in the menu inside users area. The URL is adlinks.php?action=adlink_create. It uses template adlink_create.html, these variables can be used:

#%in1%#
#%in2%# 

Content of files _in1.txt .... . This may be used to insert some information to several or all pages. In these files may be also used some variables. More info here.

#%head%#

Content of file _head2.txt. This may be used to insert some information to the begin of several or all pages. More info here.

#%site_url%#

URL of your site.

#%form%#

The form to create a new AdLink. This form is based on template adlink_create_edit_form.txt (see below).

The AdLink create form is based on template adlink_create_edit_form.txt. There are available the following variables:

#%site_url%#

URL of your site.

#%action%#
#%n%#

Hidden form fields. Don't edit these values.

#%field_categories%#

A field to select Categories. This part is based on template form_no_field.txt.

#%field_keywords%#

A field to enter keywords. This part is based on template form_field.txt.

#%field_title%#

A field to enter Title. This part is based on template form_field.txt.

#%field_url%#

A field to enter link URL. This part is based on template form_field.txt.

#%field_text1%#
.....
#%field_text10%# 

Fields to enter 1-10 text lines. You can set the number of fields. Each of these fields is based on template form_field.txt.

#%field_html%#

A field to enter link complete HTML code of the AdLink. It can be visible or invisible, depending on your configuration. This part is based on template form_detail_html.tx.

#%field_enabled%#

Checkbox to enable or disable selected AdLink. This part is based on template form_no_field.txt.

Once an AdLink has been created, it can optionally (if you selected so) send an email to admin. This email uses the email template adlink_admin.txt.
Then it shows a page which uses template adlink_edit.html. This template is also used for those users who want to edit their existing AdLinks or add new clicks to some of them. More info below.

Existing AdLinks

Existing AdLinks are available for registered users in the menu in users area, there is the link "Your links & invoices". URL of the page is link_edit.php?action=links_list. This page uses template links_for_owner_list.html. The second table on this page shows all existing AdLinks. Existing AdLinks are displayed by the variable #%adlinks%#, each line in this list is based on template links_for_owner_list_adlink.txt. This template contains the following variables:

#%site_url%#

URL of your site.

#%n%#

Number of an AdLink

#%enabled%#
#%approved%#

The info if the AdLink is enabled by owner and approved by admin.

#%c_total%#

Total number of ordered clicks.

#%c_now%#

Total number of unused clicks.

#%preview%#

AdLink preview.

#%hide_delete_begin%#
#%hide_delete_end%# 

If you don't allow your users to delete their AdLinks, these variables hides the option to delete it.

Each line in the table above contains a link to edit a selected AdLink. The URL of the edit page is adlinks.php?action=adlink_edit&n=ADLINK_NUMBER. There is also the option to manage clicks of the selected AdLink. This page uses template adlink_edit.html. There are available these variables:

#%in1%#
#%in2%# 

Content of files _in1.txt .... . This may be used to insert some information to several or all pages. In these files may be also used some variables. More info here.

#%head%#

Content of file _head2.txt. This may be used to insert some information to the begin of several or all pages. More info here.

#%site_url%#

URL of your site.

#%preview%#

AdLink preview.

#%form%#

AdLink edit form. This form uses template adlink_create_edit_form.txt (see above).

#%c_now%#

Total number of unused clicks.

#%currency%#

Currency used for all payments in your system.

#%price_min%#
#%price_max%#

Allowed price range. These limits are editable in configuration of public forms.

#%funds_balance%#

Balance of funds (money) on users account. These funds can be used to order new clicks for the selected AdLink.

#%price%#

Currently valid price per click. This price is editable by AdLink owner. The edit field is available only if there is not any clicks balance available.

#%hide_delete_begin%#
#%hide_delete_end%# 

If you don't allow your users to delete their AdLinks, these variables hides the option to delete it.

In the form above is also the option to add new clicks to the selected AdLink. There are two options:

  • Users can use the funds (money) they purchased earlier by using a package. In this case they use the first field below "Add Clicks by Using Funds on Your Account". They have to enter the number of clicks to add and a price for each click (if it hasn't been entered earlier). The clicks are instantly added to the selected AdLink.
  • Users can purchase new clicks by using a credit card or Paypal. In this case they use the second field "Add Clicks & Pay by PayPal or A Credit Card".  They have to enter the number of clicks to add and a price for each click (if it hasn't been entered earlier).  An article about the payment process is here.
