Extra categories of blogs - blogs.php

Extra categories are New Blogs, Popular Blogs, Editor Picks and Top Rated Blogs. Default URL's of these categories are as follows:
New Blogs blogs.php?action=new
Editor Picks blogs.php?action=pick
Top Rated Blogs blogs.php?action=top_rated
Popular Blogs blogs.php?action=popular

The template used for these pages is category_special.html. There are available the following variables:

#%in1%#
#%in2%# 

Content of files _in1.txt .... . This may be used to insert some information to several or all pages. In these files may be also used some variables. More info here.

#%head%#

Content of file _head2.txt. This may be used to insert some information to the begin of several or all pages. More info here.

#%site_url%#

URL of your site

#%title%#

Page title. You can define these titles in Administration => Messages.

#%items%#

List of blogs. Each blog is based on template blog_a.txt.
