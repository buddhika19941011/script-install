Link Up Gold - Introduction

Link Up Gold is an advanced script for running your own web directory. It supports an unlimited number of categories in an unlimited number of levels. Most pages is generated online, the data are always fresh. There are 5 main groups of content: links, articles, blogs, news from RSS sources and videos from youtube.com.

All public pages are based on templates, this feature allows you to edit them, remove unwanted features etc. Templates are normal HTML or text files and can be edited in your favorite HTML or text editor. These templates contain special tags (variables) which look like #%someword%#. These tags are replaced by some values (link Link Title, Description, any numbers and so on). In this Manual you can find a detailed description of each template.

If you don't want to generate pages on-the-fly but save them as common HTML files (it may be useful if you have a high traffic site, php scripts need more server resources than html files), you will need a special plugin which may be purchased extra in our owner's area. This plugin also has another option - not to create static pages but use Apache Rewrite module. In this case all pages stay dynamic but your URL's will look like they were. More info regarding this plugin may be found here.

Links

Links can be added by users, admin, there is also the option to import links from Google search results and from Dmoz directory. You also can sell advertising links. These are listed at the top in categories and search results and also can be highlighted.
Titles and short descriptions are listed in categories, each link has its own details page with all its details. Each link can include one or more images. One of them is by default displayed in categories, all images are displayed on article detail pages. 

Articles

Articles can be added by users and admin. Titles and short descriptions are listed in categories, each article has its own details page with the complete text. Each article can include one or more images. One of them is by default displayed in categories, all images are displayed on detail pages.

Blogs

Blogs can be added by users and admin. Titles and short descriptions are listed in categories, each blog article has its own details page with the complete text. Each article can include one or more images. One of them is by default displayed in categories, all images are displayed on detail pages.

Videos

Videos can be automatically imported from Youtube. Each category allows you to define keywords, these are used to import videos to that category. It also loads previews of videos, these are displayed in categories. The complete videos together with all details are displayed on video detail pages.
New videos can be imported automatically, you get a new videos without any exertion. You also can manually enter new videos in administration.

News

The system can load news from RSS sources available on the Internet. Each category of news can be automatically updated many times a day, without your assistance.
Categories of links, articles and videos also can have their own news imported from RSS sources. It's however recommended to import this content only to categories for news to make the directory clear.
