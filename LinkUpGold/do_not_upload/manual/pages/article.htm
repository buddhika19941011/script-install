Article details page - article.php

This script is used to show detailed info about a single article. This script has to be called as article.php?n=article_NUMBER where article_NUMBER is number of the selected article. To show this page is used template article_details.html by default, however you can create any number of your own templates and in setting of each category set up different templates for different categories. Names of your own templates must look like: 'article_details*.html' where the * will be any English letters, numbers and/or this character: _. The maximum allowed number of characters in names of your own templates is 40, incl. dot and extension. If you will create your own templates, make sure the default template article_details.html always exists, or you receive an error. Available variables:

#%in1%#
#%in2%# 

Content of files _in1.txt .... . This may be used to insert some information to several or all pages. In these files may be also used some variables. More info here.

#%head%#

Content of file _head2.txt. This may be used to insert some information to the begin of several or all pages. More info here.

#%site_url%#

URL of your site

#%n%#

Number of the article.

#%title%#

Title of the article.

#%description%#

Subtitle of the article.

#%text%#

Text of the article.

#%tags%#

Keywords of the article and category name where is listed this article. Each of these tags contain a link to search results for it.

#%pages_list%#

If the article has multiple pages, it shows a list of them.

#%user_defined%#

User Defined Items. Each line of this variable is based on template user_item_listing.txt.

#%user_item_28%#

If you, for some reasons, don't want to display all available User Defined Items, use this variable to display each individual item. This is only an example, replace the number 28 by real number of the item you want to display. These numbers can be found in Configuration => User Defined Items.

#%user_item_name_28%#

Use this variable to display name (label) of an individual user defined item.

#%user_item_value_28%#

Use this variable to display value of an individual user defined item.

#%created%#

Date created.

#%updated%#

Date updated.

#%rating%#

Current rating of the article.

#%rateicon%#

Relevant icon of the actual rating. These icons has to be stored in directory 'images' and their names must be '1.gif' for rating 0.5, '2.gif' for rating 1, ...... '10.gif' for rating '5'. If you will use other file formats (jpg or png) for these images, you have to set it in your Configuration (in the Administration).

#%votes%#

Number of votes for the article.

#%hits%#

Number of readers if the article.

#%hits_m%#

Number of readers if the article the current month.

#%pictures_gallery%#

If the article has images, this variable displays previews of all of them.

#%map%#

If the article has an included map, it shows it.

#%comments%#

Number of comments received by the article.

#%show_comments%#

Comments for the selected article. Each comment is based on template comment.txt.

#%more_items%#

List to other articles entered by the same person (email address).

#%next_category%#
#%previous_category%#
#%next%#
#%previous%#

Links to next/previous articles and next/previous articles in the same category.

#%icons%#

If the article is New, Popular, Updated or Editor Pick it shows the relevant icon(s).

#%tell_friend_box%#

A form to email a friend about this article. It uses the template tell_friend.txt.

#%notes_edit_box%#

A form to enter private notes (available for registered users only). It uses the template notes_form.txt.

#%notes%#

Entered private notes (available for registered users only).

#%contact_box%#

A form to contact the link owner. It uses the template contact_form.txt.

#%add_delete_favorites%#

The option to add or remove the article from user's favorites (available for registered users only).

#%edit_link%#

A link to an article edit page. This is available for admins only.

#%hide_something_begin%#
#%hide_something_end%# 

There are multiple variables which look like this example. These variables are used to hide content which is not applicable for the current category or state. You should not edit these variables.

This script is also used to create a special page for article printing. Call it as article.php?action=print&n=article_NUMBER. The page for printing is based on template article_print.html. There are not included the common header and footer as for other pages.
